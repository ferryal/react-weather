## React Forecast Weather Web App
1.  Show 5 days forecasting
1.  Option city, user can change city

## Tech Stack

1. React
1. Redux 
1. Axios
1. NodeJs
1. NPM
1. Yarn

### Install Package

```
npm install / yarn install
```

### Setup Development Mode

```
npm start / yarn start
```
